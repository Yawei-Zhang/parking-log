## Parking Lot

### 停车
Given 一个有空位的停车场和一辆车要停的车

When 这辆车停车

Then 返回"停车成功"

---

Given 一个没有空位的停车场和一辆车要停的车

When 这辆车停车

Then 返回"停车失败"

### 取车
Given 一辆车已停在停车场的车

When 这辆车取车

Then 返回"取车成功"

---

Given 一辆未停的车

When 这辆车取车

Then 返回“取车失败”

### 动态场景

Given 一个容量为 1 的停车场

When 停一辆车，取一辆车，再停一辆车

Then 返回写着“停车成功的”停车收据

---

Given 一个容量为 2 的停车场，且有一辆别的车在停车场里

When 停一辆车

Then 返回刚才停的车
